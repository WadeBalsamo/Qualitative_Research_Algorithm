Machine Learning-Assisted Analysis for Iterative Refinement of Mindfulness-based therapy program development
To enable systematic analysis within our between-cohort refinement window, we will employ machine learning tools to accelerate qualitative data processing while maintaining methodological rigor. This approach focuses on augmenting rather than replacing clinical expertise in identifying patterns relevant to intervention refinement.
Our analytical pipeline utilizes a local instance of an open-source Whisper for automated transcription of session recordings, followed by manual verification. The verified transcripts undergo automated qualitative data coding. This process tags transcript segments according to theoretical constructs including attention regulation, experiential avoidance, metacognitive awareness, and pain reappraisal. The system generates frequency counts for each code per session, enabling rapid identification of prevalent themes, and surfacing participant quotes which best represent each theme, for verification of participant understanding of concepts.
This system can facilitate clinical review by focusing on participant challenges and therapeutic moments, allowing the clinical team to quickly identify which parts of sessions require deeper analysis. Quantitative data analysis remains straightforward, calculating weekly pain averages, practice completion rates, and simple change scores to characterize overall cohort patterns rather than individual trajectories.
Integration occurs through manual synthesis of coded themes with aggregated quantitative patterns. For each session and cohort, we generate a summary report documenting the most frequent qualitative codes, overall pain trends, and average practice adherence. This synthesis identifies potential areas for intervention refinement, such as sessions where multiple participants express similar challenges or weeks showing consistent drops in practice completion.
Between cohorts, we track changes in code frequencies and aggregate outcomes to assess whether modifications address identified challenges. This systematic documentation enables evidence-based refinement decisions while acknowledging the limitations of small-sample pattern detection. All modification decisions require clinical team consensus based on convergent evidence across data sources.
The primary value of this ML-assisted approach lies in reducing the time required for transcript coding from weeks to days, enabling systematic pattern identification across sessions, and maintaining consistent coding criteria across cohorts. By automating routine analytical tasks, the clinical team can focus on interpretation and refinement planning within the constrained timeline. This pragmatic approach balances analytical efficiency with the methodological rigor required for a peer-reviewed clinical trial, supporting our iterative convergent design for hte feasibility trial of the Move-MORE protocol.
